分数の和

与えられた分数列の和を計算せよ。

Input

入力は複数のテストケースからなる。

各テストケースは2行からなり、最初の行には10以下の正整数 n が書かれている。

テストケースの2行目には、n 個の分数 r1, r2, ... , rn が空白区切りで書かれている。各分数 ri は2つの整数 ai, bi をこの順にスラッシュ記号で連結したものとして表現される。このとき、 -109 ≤ ai ≤ 109 , 0 < bi ≤ 109 である。

テストケースの終わりはEOFで示される。

Output

各テストケースについて、S = r1 + r2 + ... + rn の値を小数で出力せよ。

ただし、S の絶対値が 109 を超える場合は、|S| = 109 とすること。符号は変化しない。

誤差は絶対誤差で 10-7 以下に抑えること。
